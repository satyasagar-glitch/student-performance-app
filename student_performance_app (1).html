About this project

This dashboard enables post-prediction analysis of student academic performance using interactive data visualization.

It identifies early academic risk, compares predicted vs actual results, and highlights key influencing factors like attendance, study hours, and internal marks.

Designed to support data-driven decision-making by educators and help students improve through clear visual feedback.

Tools: HTML · CSS · JavaScript · Chart.js
Dataset: Student performance records — Engg. Dept.
